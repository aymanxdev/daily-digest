# Daily Digest

My little corner of automation that keeps itself alive. Twice a day a GitHub
Action grabs the Hacker News top stories, once a day another grabs a cat fact —
they get committed here and published as a small searchable site.

**🔎 [Search the archive →](https://aymanxdev.github.io/daily-digest/)**
